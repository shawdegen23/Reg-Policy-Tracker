# Reg-Policy-Tracker

Live command console for California regulatory monitoring (CPUC, CEC, CARB, AQMD, Legislature) for the TEC Regulatory Policy & Strategy role.

Static single-page site — no build step. Vercel serves `index.html` at the root automatically.

## Deploy
Push to `main`; the linked Vercel project auto-builds and publishes.

## Local preview
Open `index.html` in any browser.
